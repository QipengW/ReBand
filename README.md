# ReBand: Dominant-Frequency-Anchored Spectral Reorganization for Long-Term Multivariate Time Series Forecasting
## Data setting
The ReBand model requires the input data to be in the shape of [B, N, T], where B is the batch_size, N is the number of nodes, and T is the input time step length.
## Requirement
torch==1.10.2 numpy==1.19.2 pandas==1.1.5 scikit-learn==0.24.2
